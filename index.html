About Me

Hi, I’m Rindah Agustina, a third-year Information Systems student with a strong interest in technology, design, and analysis.
For me, those three areas complement each other. Technology serves as the foundation, design as the bridge to user experience, and analysis as the basis for decision-making.
By combining them, my goal is to contribute to projects that empower people through technology and engaging design. I believe good design should not only look appealing but also solve real problems.
I’m eager to keep growing, experimenting, and collaborating to create useful and meaningful digital experiences.

Project

Makan Apa Ya?

A group project for the Human-Computer Interaction course, Information Systems undergraduate program, Faculty of Computer Science, UPN “Veteran” Jakarta. The app provides personalized food recommendations based on user preferences through a filtering system. Designed in Figma and developed using the Design Thinking approach, including user research, prototyping, and usability testing.

Redesign Website FIK

This is a group project related to the NewFIK website (upnvj.ac.id), completed as the final assignment for the Information Systems Analysis and Design course at UPN ‘Veteran’ Jakarta. Using the PIECES framework and SPSS as a statistical tool, the project evaluates the website’s quality based on user perceptions, covering aspects such as performance, usability, efficiency, and service. Together with my group members, I conducted observations, surveys, and interviews, followed by enhanced UI design, DFD and ERD modeling, as well as prototype testing using Maze.

Cakra News

“Cakra News” is a group web development project created as the final assignment for the Web Programming Practicum course in the Information Systems undergraduate program, Faculty of Computer Science, UPN “Veteran” Jakarta. I was responsible for building the front-end of several main pages, including the category list, national news, international news, and news content pages. The project was developed using HTML, CSS, and Tailwind CSS.

STUDA

“STUDA” is a personal project I developed as a desktop application based on Java to help students manage their daily tasks. The application provides features for recording and categorizing tasks into two main types: academic tasks and household tasks. By utilizing JavaFX, I designed a simple and user-friendly graphical interface that allows users to add, edit, complete, and delete tasks.
